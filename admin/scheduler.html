      📐 One-Time Canva Template Setup Guide (read once, skip forever after)
      
      
        
      Create one Canva template at 1000×1500px with named text layers. Every pin you make is just: open template → paste text into layers → swap background image → download. Under 2 minutes per pin once set up.
      
        
      
          
      
            
      Step 1
      
            
      Create the canvas
      
            
      New design → Custom size → 1000 × 1500 px. Save as "WGD Pinterest Template". This is your master — never edit the master directly, always duplicate first.
      
          
      
          
      
            
      Step 2
      
            
      Add background image layer
      
            
      Add a full-canvas image placeholder. Set it as the bottom layer. You'll swap this with your AI-generated portrait image each time. Use a dark overlay (60–70% opacity black rectangle) on top so text is always readable.
      
          
      
          
      
            
      Step 3
      
            
      Create exactly 3 text layers
      
            
      
              
        
                
      • HEADLINE — bold, large (48–64pt). Top 60% of canvas. This is the pin title.
        • 
                
      • SUBLINE — medium weight (24–30pt). Below headline. The benefit line.
        • 
                
      • URL WATERMARK — small (16pt), bottom of canvas. Your blog URL, e.g. wowglamdecor.com
        • 
              
      
            
      
          
      
          
      
            
      Step 4
      
            
      Lock the URL watermark
      
            
      Right-click the URL layer → Lock. It will never move or get deleted accidentally. Your URL stays on every single pin automatically.
      
          
      
          
      
            
      Step 5
      
            
      Per-pin workflow
      
            
      Duplicate template → swap background image → paste HEADLINE text → paste SUBLINE text → download as JPG, quality 80% → upload to Pinterest. Done. Under 2 min.
